# Proyecto-An-lisis-de-Ventas-Walmart
### Objetivo Principal
Desarrollar habilidades avanzadas en Google Sheets para análisis de datos empresariales, enfocándose en limpieza, transformación y visualización de datos para la toma de decisiones estratégicas.

### Proyecto Central: Análisis de Ventas Walmart
Contexto: Crear un resumen ejecutivo que ayude a la Dirección Comercial en decisiones de presupuesto e inventario.
Dataset: Datos de ventas semanales con información de:
Ventas por tienda y departamento
Información de feriados
Tipos y tamaños de tiendas
Categorías de departamentos
### Habilidades Técnicas Desarrolladas
Limpieza y Preparación de Datos:
Normalización de formatos (fechas, números, texto)
Estandarización con funciones UPPER(), TRIM(), CLEAN()
Creación de columnas calculadas (YEAR(), WEEKNUM())
Manejo de duplicados y valores faltantes

### Funciones Avanzadas:

Búsquedas: XLOOKUP() para enriquecer datasets
Condicionales: SUMIFS(), COUNTIFS(), AVERAGEIFS()
Texto: SUBSTITUTE(), SPLIT() para transformaciones
Fechas: Manipulación y estandarización temporal
Tablas Dinámicas:

Construcción de resúmenes multidimensionales
Campos calculados personalizados
Filtros dinámicos por año/período
Cálculos de porcentajes del total
### KPIs y Métricas de Negocio
KPI 1: Eficiencia por Metro Cuadrado

Ventas por m² = SUM(ventas_semanales) / AVERAGE(tamaño)
KPI 2: Participación por Departamento

Participación % = Ventas_Departamento / Ventas_Totales
KPI 3: Contribución de ventas por Departamento


### Visualización y Dashboard
Dashboard Interactivo:
Menús desplegables para filtrado dinámico
Tarjetas de KPIs con formato condicional
Gráficos de barras para comparaciones
Layout profesional con zonas definidas

Elementos Visuales:
Gráficos de eficiencia por departamento
Comparaciones de participación de mercado
Formato condicional para alertas
Paleta de colores coherente

### Comunicación Ejecutiva
Método C-F-I (Context-Finding-Implication):
Context: Definir el marco del análisis
Finding: Presentar hallazgos con evidencia
Implication: Traducir insights en acciones

Estructura de Reportes:
Resúmenes ejecutivos concisos
Respaldo de datos para cada conclusión
Recomendaciones accionables
Documentación de metodología
Quality Assurance (QA)
Validaciones Sistemáticas:

Verificación de integridad de datos
Documentación de transformaciones
Control de duplicados y valores nulos
Trazabilidad de cálculos
